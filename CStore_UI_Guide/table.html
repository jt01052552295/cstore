주의사항

  • input요소 등 실제 주요 HTML 사용순서는 [마크업 가이드] 페이지 참고
  • col width 대신 col style="width" 권장. HTML5 validation에서 오류로 체크됨
  • row 타입 : mBoard + type2 일 경우 th width는 고정된 값을 사용.
  • 체크박스만 셀에 삽입될경우 .chk로 width값 제어
  • 셀병합될경우 tr에 .rowMerge 추가하여 보더제어(기본은 tr 첫번째 td에 왼쪽 보더없음)
  • .eTr : tr 오버시 배경색 변경에 사용 (행을 선택할 경우만)
  • 게시판 리스트일 경우 '테이터 없음'셀 하단에 추가
  • 폼요소의 부가 설명은 .txtInfo 사용
  • td 폭을 좁게 사용해야 하는 경우 tbody에 .gCellSingle 추가 : 좌우 패딩 5px로 변경
  • .gDivision : 테이블 구분, tbody 구분 필요시 추가(상단 보더 제어)
  • 셀 내에서 버튼 정렬은 [button] 페이지를 참고한다

[gTableMerge > typeHead, typeBody] tbody height 고정

  • <div class="gTableMerge">로 typeHead,typeBody를 묶어준다.
  • thead영역을 typeHead, tbody영역을 typeBody로 구분 한다.
  • typeBody의 높이값 조정이 필요할 경우 [gHeight숫자] 클래스를 추가한다.
  • typeHead의 마지막 셀 너비를 typeBody 보다 17px 늘려준다.
  • 너비조정은 px,%,class로 모두 가능하며 상황에 맞게 선택한다.
